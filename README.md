
# 🚀 AI SaaS Platform

## 📌 Features
- Upload CSV file
- Automatic EDA (Exploratory Data Analysis)
- ML Predictions with Feature Importance
- Chat with your data using NLP
- Interactive Visualizations

## 🛠️ Tech Stack
- Python
- Gradio (UI)
- Scikit-learn (ML)
- Pandas (Data Processing)
- Matplotlib (Visualization)

## 🚀 How to Run
```bash
pip install -r requirements.txt
